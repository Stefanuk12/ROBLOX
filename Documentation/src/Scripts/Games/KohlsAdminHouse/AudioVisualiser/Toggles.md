# Toggles

#### torbiter
This feature toggles the orbiter on and off.

Usage:
```
:torbiter
```

#### tvisualise
This feature toggles the Audio Visualiser part of things.

Usage:
```
:tvisualise
```

#### looporbit
This feature toggles the loop orbit where everyone takes turns with the orbiter.

Usage:
```
:looporbit
```
